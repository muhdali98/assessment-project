# Assessment Project


## Prerequisites

Before running the project, make sure you have the following installed:

- Local development environment with Apache and MySQL (e.g. Laragon, XAMPP, WAMP)
- PHP 8.1 or higher
- MySQL 8+
- Node.js 18+
- npm
- Git

---

## Local Setup Guide

1. Clone the repo
2. Import db.sql

## Frontend Dependencies

1. Run `npm install`
2. Run `npm run dev`

### Backend

1. Navigate to the backend folder:
2. Start the PHP built-in server: `php -S localhost:8000`
